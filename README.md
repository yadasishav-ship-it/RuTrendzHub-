This repository contains a starter full-stack scaffold for Rubi Trendz Hub (saree shop). It implements:
- Product + projects flow: other shops can create "projects" with contact info (first name, phone, location) via an "Other" button.
- WhatsApp notifications to other shops (via Twilio) when a project is created.
- Project lifecycle: open -> delivered -> closed.
- Monthly sales report of sales made to other shopkeepers and project history.

Tech stack
- Backend: Node.js, Express
- Database: PostgreSQL
- Frontend: React (Vite or Create React App compatible)
- WhatsApp: Twilio WhatsApp API (example integration)
